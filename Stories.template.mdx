# Button

A reusable button component for adding interactivity to your UI.

## Functionalities
    * Triggering actions and events


## Variations
    * Primary
    * Secondary

## Usage
    `import { Button } from 'button-component';`

    `<Button onClick={handleClick} label="Test"/>`

    `<Button onClick={handleClick} secondaryBottom label="Test"/> `
